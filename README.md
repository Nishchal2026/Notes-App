# Notes-App

A modern and interactive Notes App built using HTML, CSS, and JavaScript.
This app allows users to create, edit, delete, and store notes permanently using Local Storage.

# Features

-- ➕ Add Notes
-- ✏️ Edit Existing Notes
-- 🗑️ Delete Notes
-- 💾 Notes saved in Local Storage
-- 🎨 Multiple Background Themes
-- Normal Mode
-- Dark Mode
-- Gradient Mode
-- 📦 Responsive Notes Layout
-- 🖱️ Hover Effect for Edit/Delete Buttons
-- 📋 Dialog Box UI for Adding Notes

# Technologies Used

-- HTML5
-- CSS3
-- JavaScript (Vanilla JS)
-- Local Storage API

# How It Works

-- Click the Add Note button
-- Enter title and content
-- Click Add Note
-- Your notes are stored automatically in Local Storage
-- Hover over notes to Edit or Delete them
-- Change app themes using theme buttons
